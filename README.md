## Technologies and Tools Used
- React with Typescript
- Tailwind CSS
- HyperUI CSS
- Insomnia
- PostgreSQL
- Express & Node js
- AWS RDS

## Description
A homework management app to help teachers streamline the process of collecting, tracking and managing homeworks given to classes. Teachers can easily log the submission status of students' homework easily, have an overivew of the students submission, a notes tracker to help them keep the notes during marking.
